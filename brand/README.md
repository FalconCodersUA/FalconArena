# FalconArena Brand Kit

This directory contains standalone FalconArena logo assets for browser branding, documentation, and presentations.

Files:
- `brand-mark.svg` — primary full-color brand mark based on the canonical FalconArena logo.
- `brand-mark-mono.svg` — monochrome version for neutral layouts, print, and one-color surfaces.
- `brand-mark-compact.svg` — compact small-context mark for sidebar, favicon, and tiny browser surfaces.
- `brand-mark-compact-mono.svg` — monochrome compact mark for one-color small-context use.
- `favicon.svg` — browser favicon asset optimized from the compact brand mark.

Recommended use:
- Product/browser favicon: `favicon.svg`
- In-app compact mark: `brand-mark-compact.svg`
- Docs, slides, presentations: `brand-mark.svg`
- Dark/light neutral or print contexts: `brand-mark-mono.svg`

Runtime copies of the same assets are also stored in `apps/frontend/public/`.
