# Security Policy

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

If you discover a vulnerability in FalconArena, report it privately to the project owner by email:

```text
yspersik@gmail.com
```

Include as much detail as possible:

- affected area or endpoint;
- steps to reproduce;
- expected and actual behavior;
- screenshots, logs, or proof-of-concept details if available;
- potential impact.

## Response Expectations

The maintainer will review the report, confirm the affected scope, and prioritize a fix based on severity. Security fixes may be prepared privately before public disclosure.

## Supported Version

The actively maintained version is the current `main` branch and the production deployment at:

```text
https://falconarena.live/
```

## Responsible Disclosure

Please avoid accessing, modifying, or sharing data that does not belong to you. Give the maintainer reasonable time to investigate and fix confirmed issues before public disclosure.
